# VoltGo

VoltGo is a premium smart EV charging management platform designed to help users discover stations, book charging slots, make payments, manage reservations, and receive QR-based confirmations. Administrators can oversee stations, chargers, users, bookings, payments, and analytics through a dedicated admin dashboard.

## Project Goal

This repository currently contains the planning, UI/UX, and project setup foundation for a scalable, production-ready application.
