## Design Objective

VoltGo should feel like a premium, modern, commercial EV charging platform with a smooth booking experience, polished interactions, and a consistent design system across web and mobile experiences.

## Design Principles

- Premium and minimal rather than visually noisy
- Fast and confident interactions
- Clear information hierarchy
- Calm, trustworthy, and modern visual language
- Mobile-first yet desktop-optimized
- Strong consistency between customer and admin experiences
