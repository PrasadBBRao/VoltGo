# 4. User Journey

## Primary User Journey: Book a Charging Slot

### Step 1: Discover the Product
The user lands on the VoltGo homepage and sees a polished hero section with a strong value proposition and a clear CTA.

### Step 2: Create or Sign In
The user either registers for a new account or signs in to continue. The form is simple, clear, and respectful of the user’s time.

### Step 3: Explore Nearby Stations
The user searches for nearby charging stations using location, filters, or saved preferences.

### Step 4: Review Station Details
The user opens a station page to see availability, charger types, pricing, and operating information.

### Step 5: Select a Slot
The user chooses a date and time slot. The interaction should feel as simple and confident as booking a ticket.

### Step 6: Review Booking Summary
The user reads a concise summary of the station, time, charger, and pricing before confirming.

### Step 7: Complete Payment
The user completes the payment using a smooth, frictionless flow. Confirmation is immediate and reassuring.

### Step 8: Receive Booking Confirmation
The user sees a success state with booking details and a QR confirmation. The experience feels polished and complete.

### Step 9: Arrival and Repeat Use
The user can later access booking history, save favourite stations, and repeat the experience with less effort.

## Secondary User Journey: Admin Operations

### Step 1: Admin Sign In
An administrator logs in to the admin experience.

### Step 2: Review Dashboard
The admin sees operational health, revenue stats, and recent activity.

### Step 3: Manage Stations and Chargers
The admin updates station availability, charger status, and operating details.

### Step 4: Monitor Bookings and Payments
The admin checks bookings and payment statuses to resolve issues quickly.

### Step 5: Generate Insights
The admin uses analytics dashboards to support operational decisions.

## UX Intent

The journey is designed to feel seamless, premium, and guided. Every step removes ambiguity and highlights the next best action.
