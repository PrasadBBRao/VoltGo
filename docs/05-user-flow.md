# 5. User Flow

## Primary User Flow: EV Driver

### 1. Landing Experience
The user arrives at the platform and sees a polished landing page that explains the service, highlights nearby stations, and promotes the booking experience.

### 2. Registration or Login
The user signs up or logs into an existing account. The system authenticates the session and prepares the user for station discovery.

### 3. Search for Charging Station
The user searches for a station by location, nearby area, or preferred charger type. The system returns a list or map of available stations.

### 4. Review Station Details
The user reviews station information including charger availability, connector types, pricing, opening hours, and service quality indicators.

### 5. Select Booking Slot
The user selects a preferred date and time slot based on availability. The system verifies the slot is still open before proceeding.

### 6. Payment Confirmation
The user completes the payment. On success, the system records the payment and confirms the booking.

### 7. Receive Confirmation
The user receives a booking summary with a QR code and appointment details. The confirmation is also stored in their account.

### 8. Arrival and Validation
At the station, the user presents the QR code, which is scanned and validated against the active booking.

## Admin Flow

### 1. Admin Login
An administrator accesses the admin dashboard via secure authentication.

### 2. Dashboard Overview
The admin views key metrics such as station health, active bookings, recent payments, and issues.

### 3. Manage Stations and Chargers
The admin adds or updates stations, assigns chargers, and controls availability state.

### 4. Review Bookings and Payments
The admin monitors active reservations, failed payments, and user actions.

### 5. Generate Insights
The admin uses analytics to improve station operations and identify performance trends.

## Why This Flow

The flow is designed to feel effortless for end users while being structured enough for operators to manage the service reliably. The experience is intentionally simple, with clear transitions between discovery, booking, payment, confirmation, and station use.
